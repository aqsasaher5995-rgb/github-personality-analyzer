GitHub Personality Analyzer

A Python project that analyzes GitHub user profiles using the GitHub API.

Features

- Fetch GitHub user information
- Display number of public repositories
- Show followers and following count
- Handle invalid usernames with error messages

Technologies Used

- Python
- GitHub REST API
- Requests library

Installation

Clone the repository

git clone https://github.com/aqsasaher5995-rgb/github-personality-analyzer.git

Move into the project folder

cd github-personality-analyzer

Install required library

pip install requests

Usage

Run the program

python main.py

Enter any GitHub username when prompted.

Example:

torvalds

The program will display the user's repository and follower information.
